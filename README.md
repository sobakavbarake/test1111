Currently includes:

- React Native

- Zustand
- TypeScript
- And more!

## Quick Start

The project's structure will look similar to this:

```
a0-project
├── app
│   ├── ui
│   ├── i18n
│   ├── models
│   ├── navigators
│   ├── screens
│   ├── utils
│   └── app.tsx
├── lib
│   └── apiClient.ts
│   └── [stores].ts
├── assets
│   └── images
├── test
│   ├── __snapshots__
│   ├── mockFile.ts
│   └── setup.ts
├── README.md
├── android
│   ├── app
│   ├── build.gradle
│   ├── gradle
│   ├── gradle.properties
│   ├── gradlew
│   ├── gradlew.bat
│   ├── keystores
│   └── settings.gradle
├── index.js
├── ios
│   ├── A0Project
│   ├── A0Project-tvOS
│   ├── A0Project-tvOSTests
│   ├── A0Project.xcodeproj
│   └── A0ProjectTests
├── .env
└── package.json

```
